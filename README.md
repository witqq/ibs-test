# Иснтрукция по запуску
## Требования
 - node.js v6.11.0
 - npm v3.10.10
 - не занятый порт 8080
 
## Инструкция
Склонировать проект 

`git clone https://github.com/witqq/ibs-test.git`

Перейти в директорию с проектом

`cd ibs-test`

Установить все зависимости

`npm install`

Запустить сборку 

`npm run heroku-postbuild`

Запустить приложение

`npm run start`
